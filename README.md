法森林大冒险
项目概述
本项目是基于 HTML/CSS/JS 开发的简化冒险类小游戏，核心玩法为角色移动、跳跃、释放魔法、收集水晶、击败敌人，通关全部关卡。项目采用 Spec Coding+Vibe Coding 开发模式，先明确策划规范，再优化体验细节，流程完整可直接运行。
项目基础信息
项目名称：魔法森林大冒险
技术栈：HTML5+CSS3+JavaScript
核心功能：角色移动跳跃、魔法技能、敌人 AI、水晶收集、5 个主题关卡、基础物理碰撞
源码仓库：https://github.com/juanliuming/Game-njl
运行说明
环境要求
Chrome、Edge、Firefox 等支持 HTML5 Canvas 的现代浏览器
本地运行无需额外软件
运行步骤
克隆或下载项目代码
直接双击打开 index.html 或魔法森林大冒险 - 完整独立运行版.html 启动游戏
操作说明
方向键←/→：左右移动
方向键↑/ 空格键：跳跃
X 键：释放火球术
C 键：释放冰锥术
通关条件：通过 5 个关卡，收集足够水晶
失败条件：生命值为 0

AI 开发流程
Spec Coding 阶段
分析小游戏平台冒险类产品，提取玩法共性
量化核心循环、操作、系统设计指标
通过 Coze 生成游戏策划案
使用 Gemini、豆包优化 Prompt
输出 Trae 可执行的开发规范
Vibe Coding 阶段
Trae 依据策划案生成游戏 Demo
优化新手引导、过关提示、难度曲线
修复跳板碰撞、魔法值消耗、敌人 AI 异常
调整 UI 布局与视觉效果
设计思路
核心设计：水晶收集 + 关卡通关双重目标，技能差异化，敌人类型区分
关卡设计：难度渐进，主题差异化，平台与陷阱组合
技术实现：面向对象封装，Canvas 渲染，轻量物理碰撞
