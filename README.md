# 🌲 魔法森林大冒险

## 📖 项目概述
本项目是基于 **HTML / CSS / JavaScript** 开发的简化冒险类小游戏。  
核心玩法包括：角色移动、跳跃、释放魔法、收集水晶、击败敌人并通关关卡。

项目采用 **Spec Coding + Vibe Coding** 开发模式：  
先明确策划规范，再优化游戏体验，流程完整，可直接运行。

---

## 🧾 项目基础信息
- **项目名称**：魔法森林大冒险  
- **技术栈**：HTML5 + CSS3 + JavaScript  
- **源码仓库**：👉 https://github.com/juanliuming/Game-njl  

---

## 🚀 核心功能
- 🧍‍♂️ 角色系统：移动、跳跃、生命值、魔法值管理  
- 🔥 魔法系统：火球术（攻击）、冰锥术（控制）  
- 👾 敌人系统：巡逻敌人、陷阱敌人 AI  
- 🗺️ 关卡系统：5 个主题关卡，难度递进  
- ⚙️ 互动机制：冰块跳板、地刺陷阱  
- 💎 收集系统：水晶收集目标
