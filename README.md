# 地牢探险者 - 冒险类小游戏

## 游戏简介
《地牢探险者》是一款基于HTML、CSS和JavaScript开发的简化版冒险类小游戏。玩家将扮演一名冒险者，在地牢中探索、战斗、收集物品，最终目标是到达第10层地牢并获得胜利。

## 如何运行游戏

### 方法一：直接打开HTML文件
1. 下载项目文件到本地
2. 找到项目目录中的 `index.html` 文件
3. 双击打开该文件，游戏将在默认浏览器中运行

### 方法二：使用本地HTTP服务器（推荐）
1. 确保您的计算机已安装Python或Node.js
2. 打开命令行终端，进入项目目录
3. 使用Python启动HTTP服务器：
   ```bash
   python3 -m http.server 8000
   ```
   或使用Node.js的http-server模块：
   ```bash
   npx http-server -p 8000
   ```
4. 在浏览器中访问 `http://localhost:8000`

## 制作过程与设计思路

### 1. 游戏核心机制设计

#### 主题选择
选择了经典的地牢探险主题，因为它具有明确的目标（到达深层地牢）和丰富的游戏元素（战斗、探索、收集），适合制作简化版冒险游戏。

#### 角色状态系统
设计了基本的角色属性：
- **生命值(HP)**：角色的健康状态，降为0时游戏结束
- **攻击力(ATK)**：影响战斗中对敌人的伤害
- **防御力(DEF)**：减少战斗中受到的伤害
- **金币(Gold)**：用于购买物品
- **钥匙(Keys)**：用于进入下一层地牢
- **层数(Level)**：当前所在地牢层数，到达10层获胜

#### 随机事件系统
为了增加游戏的可玩性和随机性，设计了以下随机事件：
- **战斗**：与随机生成的敌人战斗
- **宝箱**：获得随机物品（金币、钥匙、生命药水、属性提升）
- **商店**：可以购买各种物品
- **陷阱**：受到随机伤害
- **休息**：恢复生命值

#### 战斗系统
采用简单的回合制战斗系统：
- 玩家先攻击，然后敌人攻击
- 伤害计算：伤害 = 攻击者攻击力 - 防御者防御力（最低为1）
- 战斗胜利：获得金币
- 战斗失败：游戏结束
- 逃跑机制：有50%的概率成功逃跑

#### 胜利与失败条件
- **胜利条件**：到达第10层地牢
- **失败条件**：角色生命值降为0

### 2. 界面设计

#### 布局结构
采用了清晰的分区布局：
- **游戏标题区**：显示游戏名称
- **角色状态面板**：显示角色的各项属性
- **游戏信息区**：显示当前事件和游戏状态
- **敌人信息面板**：战斗时显示敌人状态
- **控制按钮区**：提供游戏操作按钮

#### 视觉设计
- 使用深色主题，营造地牢的神秘氛围
- 不同功能的按钮使用不同颜色，增强用户体验
- 响应式设计，适配不同屏幕尺寸

### 3. 技术实现

#### HTML结构
- 使用语义化标签组织页面结构
- 为动态元素添加ID，便于JavaScript操作

#### CSS样式
- 使用Flexbox和Grid进行布局
- 添加过渡效果，提升交互体验
- 实现响应式设计，适配移动设备

#### JavaScript逻辑
- 使用对象管理角色和敌人状态
- 采用函数模块化设计，便于维护和扩展
- 使用事件驱动的交互方式
- 实现随机事件系统，增加游戏可玩性

### 4. 游戏平衡性考虑

- 敌人属性随地牢层数增加而增强
- 每5层提升玩家基础属性，保持游戏挑战性
- 随机事件概率经过调整，确保游戏节奏合理
- 战斗难度适中，既有挑战性又不会过于困难

## 游戏操作指南

1. 点击"开始探索"按钮进入地牢
2. 探索过程中会遇到随机事件
3. 战斗时可以选择"攻击"或"逃跑"
4. 收集金币和钥匙，提升角色属性
5. 使用钥匙进入下一层地牢
6. 到达第10层获胜，生命值降为0失败
7. 可以随时点击"重新开始"按钮重新游戏
