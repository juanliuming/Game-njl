# 魔法森林大冒险 - 运行与开发文档

## 一、如何运行游戏

### 1. 环境要求
- **浏览器**：支持HTML5 Canvas的现代浏览器（Chrome、Firefox、Safari、Edge等）
- **本地服务器**：需要本地HTTP服务器来运行游戏（避免跨域问题）

### 2. 启动步骤

#### 方法1：使用Ruby内置服务器（推荐）
1. 打开终端（Terminal）
2. 进入游戏目录：
   ```bash
   cd /Users/kylinni/Desktop/冒险类小游戏
   ```
3. 启动服务器：
   ```bash
   ruby -run -e httpd . -p 8111
   ```
4. 在浏览器中访问：
   ```
   http://localhost:8111
   ```

#### 方法2：使用Python内置服务器
1. 打开终端（Terminal）
2. 进入游戏目录：
   ```bash
   cd /Users/kylinni/Desktop/冒险类小游戏
   ```
3. 启动服务器（Python 3）：
   ```bash
   python3 -m http.server 8111
   ```
   或（Python 2）：
   ```bash
   python -m SimpleHTTPServer 8111
   ```
4. 在浏览器中访问：
   ```
   http://localhost:8111
   ```

### 3. 游戏操作
- **方向键 ←→**：左右移动
- **方向键 ↑**：跳跃（支持二段跳）
- **X**：火球术（直线远程攻击，消耗10魔法值）
- **C**：冰锥术（带角度远程攻击，消耗15魔法值）

## 二、项目结构

### 1. 文件结构
```
魔法森林大冒险/
├── index.html          # 游戏主HTML文件
├── style.css           # 游戏样式文件
```

### 2. 文件说明

#### index.html
- **功能**：游戏的HTML结构，包含画布、UI元素和各种界面
- **主要内容**：
  - 游戏画布（gameCanvas）
  - 玩家状态面板（生命值、魔法值、水晶数量、关卡信息）
  - 新手引导界面（introScreen）
  - 游戏结束界面（gameOverScreen）
  - 暂停界面（pauseScreen）

#### style.css
- **功能**：游戏的样式定义，控制UI外观
- **主要内容**：
  - 全局样式（字体、颜色、布局）
  - 游戏容器样式
  - 玩家状态条样式（生命值、魔法值）
  - 技能栏样式
  - 各种界面样式（新手引导、游戏结束、暂停）

#### game.js
- **功能**：游戏的核心逻辑实现
- **主要内容**：
  - Game类：游戏主控制器
  - Player类：玩家角色
  - Enemy类：敌人角色（移动型和静止型）
  - Platform类：平台元素
  - Crystal类：水晶收集物
  - Projectile类：投射物（火球、冰锥）
  - Spike类：地刺陷阱
  - 关卡创建函数（createLevel1到createLevel5）
  - 碰撞检测系统
  - 物理系统（重力、摩擦力）

## 四、设计思路

### 1. 游戏核心设计
- **双重目标系统**：收集水晶和击败敌人，增加游戏深度
- **魔法技能差异化**：火球术（低消耗、直线攻击）和冰锥术（高伤害、带角度），提供策略选择
- **敌人类型多样化**：移动型（随机巡逻）和静止型（远程攻击），增加战斗乐趣

### 2. 关卡设计思路
- **渐进式难度**：从简单到复杂，逐步引入新机制
- **主题差异化**：每个关卡有独特的视觉主题和游戏机制
- **平台与陷阱结合**：普通平台、冰块跳板和地刺陷阱，创造多样化的挑战

### 3. UI设计思路
- **信息清晰**：所有关键信息（关卡、水晶、敌人、操作）一目了然
- **视觉层次**：使用颜色和大小区分不同类型的信息
- **操作友好**：操作提示清晰，技能冷却直观显示
- **美观协调**：统一的UI风格，良好的视觉体验

### 4. 技术实现思路
- **面向对象设计**：使用类封装不同游戏对象，提高代码复用性和可维护性
- **Canvas渲染**：高效的2D图形渲染，确保游戏流畅运行
- **物理系统**：简化的重力和碰撞检测，提供真实的游戏体验
- **模块化开发**：将游戏功能划分为多个模块，便于开发和调试

## 五、游戏特色功能说明

### 1. 敌人AI系统
- **移动型敌人**：随机巡逻，有巡逻范围限制，定期改变方向
- **静止型敌人**：待在原地，当玩家进入攻击范围时发射远程攻击

### 2. 特殊平台机制
- **冰块跳板**：玩家站在上面30帧后碎裂，碎裂后玩家自然下落
- **地刺陷阱**：玩家接触后受到伤害并被击退

### 3. 魔法技能系统
- **火球术**：直线远程攻击，适合远距离攻击
- **冰锥术**：带角度远程攻击，适合攻击高处或远处的敌人

### 4. 关卡主题系统
- 每个关卡有独特的背景和视觉风格
- 关卡名称显示在左上角，增强沉浸感

《魔法森林大冒险》是一个完整的2D横版动作冒险游戏，通过HTML5 Canvas、JavaScript和CSS实现。游戏包含5个主题关卡、多种魔法技能、多样化敌人和特殊平台/陷阱。项目结构清晰，代码组织合理，便于维护和扩展。
